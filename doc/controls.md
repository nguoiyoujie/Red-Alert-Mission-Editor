### Controls
<br>

|Action       |Means  
:-------------|:-------------------------------------
|`LMB`        |Left Mouse Button (Click)
|`LMD`        |Left Mouse Button (Click & Drag)
|`RMB`        |Right Mouse Button (Click)
|`RMD`        |Right Mouse Button (Click & Drag)
|`MMD`        |Middle Mouse Button (Click & Drag)
|`MSW`        |Middle Mouse Scroll Wheel (Scroll)

<br>

<b>Select Mode</b>
(When object list is blank)

<samp>

|Action         |Interpretation
:---------------|:-------------------------------------
|`LMB`          |Select cell
|`LMD`          |Select rectangle of cells
|`Shift + LMB`  |Add/Remove cell to selection
|`Shift + LMD`  |Add/Remove rectangle of cells to selection
|`RMB`          |Pick object on cell and change mode to what is picked
|`Ctrl + C`     |Copy selected cells (and objects on cells)
|`Ctrl + V`     |Paste selected cells relative to the first cell in selection
|`Del`          |Delete objects in selected cells
|`Shift + Del`  |Clear terrain in selected cells

</samp>


<b>Object Mode</b>
(When object list is filled)

<samp>

|Action                |Interpretation
:----------------------|:-------------------------------------
|`LMB`                 |Place object on cell
|`Ctrl + LMB`          |Remove object(s) of same type on cell
|`Shift + LMB`         |Paint object on cell
|`Ctrl + Shift + LMD`  |Paint-remove object(s) of same type on cell
|`Esc`                 |Return to Select Mode
|`RMB`                 |Pick object on cell and change mode to what is picked

</samp>

|Action                |Interpretation

|`Alt + 4`             |Smudge Mode

|`Alt + Minus`         |Cell Trigger Mode
|`Alt + Del`           |Select Mode
|`Ctrl + Z`            |Undo last edit (including edits in Editor dialog menus e.g. Triggers)
|`Ctrl + Y`            |Redo edit
|`Ctrl + M`            |Open Map Attributes Dialog
|`Ctrl + Shift + M`    |Open Shift Map Dialog

|`Ctrl + E`            |Open Editor Dialog > Teams

</samp>
